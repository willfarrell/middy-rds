# Contributing

In the spirit of Open Source Software, everyone is very welcome to contribute to this repository. Feel free to [raise issues](https://github.com/willfarrell/middy-rds/issues) or to [submit Pull Requests](https://github.com/willfarrell/middy-rds/pulls).

## Development Setup

```bash
git clone https://github.com/willfarrell/middy-rds.git
cd middy-rds
npm install
npm test
```

## Commit Format

This project uses [Conventional Commits](https://www.conventionalcommits.org/).

## Code Style

Code is formatted and linted using [Biome](https://biomejs.dev).
